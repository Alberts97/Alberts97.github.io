LeetCode 18. 四数之和

发表于 2020-02-12 | 更新于 2021-06-30 | 分类于 LeetCode

题意

从数组中找出满足和为target的四元组。

思路

  • 想法1：暴力，$O(n^4)$。
  • 想法2：排序 + 指针。思路类似前面的三数之和，只不过这里的指针多了一个，后两个指针相遇时第二个指针后移一个单位。时间复杂度：$O(n^3)$，空间复杂度：$O(1)$。

代码

总结

排序 + 指针！
